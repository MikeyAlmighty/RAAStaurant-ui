## Usage

```ts
import {Button} from "@MikeyAlmighty/raastaurant-ui";
import {ColorType} from "./constants";

const Example = () => {
    return (
        <Button
            size = {"small"}
            text = {"Button"}
            onClick = {()=>console.log("Clicked")}
            buttonType={ColorType.PRIMARY}
        />
)
    ;
};

export default Example;
```

#### Arguments

- **text** _`() => void`_ - A string that represents the text content of the button.
- **buttonType** - `Primary | Secondary | Tertiary` indicating whether the button should have a the required styling. Typically, a primary
button stands out as the main action in a user interface.
- **disabled** - **Boolean** indicating whether the button should be disabled or not. When disabled, the button cannot be
clicked or interacted with.
- **onClick** - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the
click event on the button element.
