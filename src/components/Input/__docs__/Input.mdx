## Usage

```ts
import {Input} from "@MikeyAlmighty/raastaurant-ui";

const ExampleInput = () => {
    return (
        <Input
          name={name}
          disabled={true}
          errorMessage={errors.name}
          defaultValue = {"DeadPool"}
        />
    );
};

export default ExampleInput ;
```

#### Arguments

- **name**  - A string that represents the label of the input.
- **defaultValue**  - A string that represents the `defaulValue` of the **input**.
- **errorMessage**  - A `FieldError` that represents the **errorMessage** of the **input** returned from `react-hook-form`.
- **disabled** - Boolean indicating whether the input should be disabled or not. When disabled, the input cannot be
changed.
